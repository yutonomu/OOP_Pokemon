## 📚 このプロジェクトについて

このリポジトリは、**株式会社ドボクリエイトのiOSアプリ研修プログラム**として作成されたオブジェクト指向プログラミング学習用のサンプルコードです。

ポケモンバトルという馴染みのあるテーマを通じて、以下のOOPの重要な概念を実践的に学びます：

- 📦 **クラスとインスタンス**
- 🔒 **カプセル化**
- 🔗 **継承**
- 🎭 **ポリモーフィズム**
- 📋 **インターフェース（プロトコル）**

## 🎯 学習目標

1. **オブジェクト指向の基本概念を理解する**
2. **Swiftでクラス設計ができるようになる**
3. **インターフェースを使った柔軟な設計を体験する**
4. **実際に動くiOSアプリを作成する**
